# 課程目的 #
入門學習 JavaScript 程式及 FHIR 健康醫療資訊標準；學會使用 JavaScript 程式處理 FHIR 病人、問題狀況、生理量測、檢驗檢查結果等資訊，學會JavaScript 程 式 HTML FHRM 表單輸入、資料表格呈現等功能，並體驗設計醫護表單使用介面。

# 上課方式 : 視訊討論及線上教材 #
